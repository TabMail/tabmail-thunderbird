# Welcome Wizard Assets

Place your images/videos here. The welcome wizard will automatically use them.

## Required Assets

### Logo
- `logo-light.svg` - Logo for light mode (or .png)
- `logo-dark.svg` - Logo for dark mode (or .png)

### Theme Previews (Appearance > Theme step)
- `theme-light.png` - Screenshot showing light theme
- `theme-dark.png` - Screenshot showing dark theme  
- `theme-system.png` - Screenshot showing system/auto theme

### View Mode Previews (Appearance > View Mode step)
- `view-relaxed.png` - Screenshot of relaxed/card view
- `view-compact.png` - Screenshot of compact/table view

### Tag-Based Sorting Previews (legacy)
- `example-tagsort-enabled.webp` - Screenshot showing inbox with tag-based sorting enabled (light mode)
- `example-tagsort-disabled.webp` - Screenshot showing inbox with tag-based sorting disabled (light mode)
- `example-tagsort-enabled-dark.webp` - Screenshot showing inbox with tag-based sorting enabled (dark mode)
- `example-tagsort-disabled-dark.webp` - Screenshot showing inbox with tag-based sorting disabled (dark mode)

### Feature Screenshots (Features section)
- `feature-popup.png` - Screenshot of the popup menu
- `feature-chat.png` - Screenshot of the chat interface
- `feature-templates.png` - Screenshot of the templates page

## Supported Formats
- Images: `.png`, `.jpg`, `.jpeg`, `.svg`, `.webp`
- Videos: `.mp4`, `.webm` (future support)

## Recommended Sizes
- Logo: 200px wide (SVG preferred for crispness)
- Theme/View previews: 300x200px
- Feature screenshots: 400x250px

## Notes
- If an asset file doesn't exist, a placeholder will be shown
- Dark mode logo is used when system is in dark mode
- All paths are relative to the `welcome/assets/` folder
